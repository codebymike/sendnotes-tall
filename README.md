# SendNotes

## Experiment using the TALL stack (Tailwind Alpine Livewire & Laravel)

### Using
- Laravel 11.x
- Livewire + Volt
- Tailwind + WireUI
- Sqlite


### Laravel Features
- Factory / Seeder
- Jobs + Queues
- Commands
- Policies


### Instructions
```
composer install && npm i
php artisan migrate
php artisan db:seed
npm run dev
```
